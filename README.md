## 📁 How to Add Fan Art

To add new fan art to the website carousel without touching any code:
1. Upload your image (`.png`, `.jpg`, or `.gif`) directly into the `/fanart` folder.
2. Name the file using dashes for spaces (e.g., `Cool-Drawing-by-Viewer.png`). The site will automatically convert this into the caption **"Cool Drawing by Viewer"** on the page!
